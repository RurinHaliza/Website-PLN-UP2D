# PLN UP2D Monitoring Website

Aplikasi internal berbasis web yang dikembangkan selama program magang di PLN UP2D.
Project ini digunakan untuk membantu proses monitoring data operasional dengan menambahkan beberapa fitur pengelolaan data agar proses monitoring menjadi lebih efektif.

Selama pengembangan project ini, saya berkontribusi dalam penambahan fitur monitoring data, filter data, dan export data ke format Excel.

## Features

* Monitoring data beban listrik (harian, mingguan, tahunan)
* Filter data beban berdasarkan kebutuhan monitoring (harian, mingguan, tahunan) 
* Export data ke format Excel
* Dashboard berbasis web
* Pengelolaan data yang lebih terstruktur
* Interface yang mempermudah proses monitoring internal
